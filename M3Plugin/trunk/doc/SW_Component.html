SW_Component

Description of a SW component that can be instanciated to create the SW platform

General Attributes :

  • name      [Mandatory]: Name of the SW component 
    • 

  • type      [Mandatory]: Type of SW component. 
	      - Possible values: OS, middleware 
    •

    Structure:

    < SW_Component name="OS_1" type="OS"/ >
