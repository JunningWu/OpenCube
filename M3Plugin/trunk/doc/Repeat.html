Repeat

Allows repeating groups of clauses. This is specially useful when combined with the XML System Configuration file. This allows defining a variable number of elements in the platform, as number of nodes, number of processors, ...

Attributes:

  • number [Mandatory]: Name to number of times the internal elements must be repeated
  • index [Optional]: Letter to be replaced by the instance number when preceded by '%' (See the example below)
  • init [Optional]: Indicates the value of “index” the first time the Repeat clause is applied. If it is not specified, “index” starts with “0”

    • Structure:

    < Repeat number="3" index="i" init="1" >	
	< Component_type name="name%i" >
	< Repeat number="_repeat_times" index="j" >	
		< Component_type name="name_%i_%j" >
	< / Repeat >	
< / Repeat >
