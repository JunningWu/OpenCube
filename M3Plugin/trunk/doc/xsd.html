XML System Description file

The XML System Description allows describing the system to be modeled. HW and SW components, HW and SW instances, architecture and task allocation is described in this file.

File structure:

< Description >	
	< HW_Platform >
		< HW_Components >
		< HW_Architecture >
		< Computing_Groups >
	< / HW_Platform >

	< SW_Platform >
		< SW_Components >
		< SW_Architecture >
	< / SW_Platform >

	< Application >
		< Functionality >
		< Allocation >
	< / Application >

	< Simulation >
		< Implementation >
		< Variable >
	< / Simulation >
< /Description  >

Additional clauses:

< Repeat >: used to indicate that some parts of the description has to be repeated 'n' times.
