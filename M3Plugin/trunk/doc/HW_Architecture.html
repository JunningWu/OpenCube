HW_Architecture

Contains the architectural description of the HW platform. The HW elements instanciated and its connections, as well as the hierarchy are described.

Attributes:

  • name [Optional]: Name to identify the HW platform (See Implementation)

    • Structure:

    < HW_Architecture name="name" >	
	< HW_Instance >
	< HW_Connection >
	...
< / HW_Architecture >
