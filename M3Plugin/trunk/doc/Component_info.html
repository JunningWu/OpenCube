Struct xml_component_info

Structure that contain all the parameters of a HW Instance.

It contain the parameters specified in the HW_Instance. Otherwise, the base HW_Component values are used.

General Attributes :

  • Name               Type   Description 
    • 

  • name               char*: Name of the HW_Component 
    • 

  • category           char*: Cathegory of component. 
    • 

  • type               char*: Type of Component. 
    • 

  • mem_size           int:   Amount of memory associated to the HW component in the memory map 
    • 

  • latency            float: Component latency 
    • 

  • frequency          float: Component frequency. 
    • 

  • width              int:   Component interface width.  
    • 

  • area               float: HW area required by this component. 
    • 

  • static_power       float: Mean power consumed when no bus events are received.  
    • 

  • read_energy        float: Energy consumed when a bus read event is received. In caches it is the miss energy
    • 

  • write_energy       float: Energy consumed when a bus write event is received. 
    • 

  • read_size_energy   float: Variable energy consumed when a bus read events is received, depending on the buffer size. 
    • 

  • write_size_energy  float: Variable energy consumed when a bus write events is received, depending on the buffer size. 
    • 

  • component_specific char*: Additional parameters required for a non-default SCoPE component. 
                              It is provided to the init_func function as char*.
