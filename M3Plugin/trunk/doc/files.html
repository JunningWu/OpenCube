Internal structure of the plug-in

Directories

src Sources of the plug-in

examples Examples directory

doc Directory with the documentation

build Directory where the library is placed when the plug-in is built

Source files

SCoPE link

uc_load_xml.cpp Main file; contains the sc_main, which calls the functions to read the xml files and to create the system model.

uc_create_xml_platform.cpp Functions to create in SCoPE the model of the system described in the XML files

XML files management

xml_main.c Main functions to read the XML System Description file.

xml_configuration_file.c Functions to read the XML System Configuration file.

xml_metric_definition.c Functions to read and manage the XML Metrics Definition file.

xml_metric_output.c Functions to create the output file with the required metrics.

Files to decode the XML System Description file

xml_if.c Decode the attributes of the clauses in the System Description file.

xml_input.c Organizes the decoded attibutes.

XML generic management

xml_list.c Auxiliar file implementing a simple list.

xml_reader.c Functions to read the XML files and extract the clauses and attributes. Can be replaced by the expat library

xml_hierarchy.c Create the XML structures to maintain the XML file hierarchy. Used to read the System Description file
