HW_Connection

Indicates that a predefined instance will be also connected at this point of the platform

General Attributes :

  • name      [Mandatory]: Name of the HW instance 
    • 

  • instance  [Mandatory]: Name of the HW instance to be connected 
    •  

  • master_slave [Optional]: It indicates if the component must be connectedas "master", "slave" or "both". 
		Slave is selected by default. Instance defined values for address, irq, latency and others are applied   
    •

    Structure:

    < HW_Connection name="net_if_1_connection" instance="my_net_if" / >
