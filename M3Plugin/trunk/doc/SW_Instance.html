SW_Instance

Description of an instanciation of a SW component to create the SW platform

General Attributes :

  • name        [Mandatory]: Name of the SW component 
    • 

  • component   [Mandatory]: Name of SW component to be instanciated
    • 

  • hw_resource [Mandatory]: Name of HW resource where the SW component will be executed. It can be the name of a HW_Instance or a Computing_Group
    •

    Structure:

    < SW_Instance name="OS_1" type="OS"/ >
